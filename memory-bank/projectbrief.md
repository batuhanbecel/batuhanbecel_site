# Project Brief: Batuhan Becel Portfolio

## Overview
Personal portfolio website for Batuhan Becel, a professional retoucher based in Istanbul, TÃ¼rkiye.

## Core Requirements
- Modern, minimal, editorial design
- Fast, distraction-free experience
- Professional yet creative presentation
- Target audience: Creative directors, agencies, brands, art directors

## Tech Stack
- **Framework**: Next.js 16.1.3 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion (subtle)
- **Deployment**: Vercel
- **Containerization**: Docker

## Key Features
1. Floating vertical navigation (left side, icon-based)
2. Single-page main site with sections: Hero, About, Experience, Education, Skills, Portfolio preview
3. Separate /portfolio page with masonry grid and lightbox
4. Auto-loading images from /public/portfolio-images
5. Social media links (Behance, Instagram, LinkedIn)

## Constraints
- No contact form
- No blog
- No filters on portfolio
- SEO: Medium priority
